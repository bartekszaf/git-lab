Bootstrap

Bootstrap to potężny zestaw narzędzi i gotowych elementów do tworzenia responsywnych interfejsów graficznych aplikacji internetowych i stron www.
Aby dodać Bootstrapa do projektu, trzeba dodać poniższe linijki:
HTML
Copy
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous"> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
​

Proszę zapoznać się z podstawowymi elementami bootstrapa.

System Siatki (Grid System)

Bootstrap wykorzystuje system siatki do tworzenia układów stron. Jest on oparty na kontenerach, wierszach i kolumnach, które pomagają w organizacji zawartości.
Przykład:
HTML
Copy
<div class="container"> <div class="row"> <div class="col-sm-4"> <p>Jestem kolumną 1</p> </div> <div class="col-sm-4"> <p>Jestem kolumną 2</p> </div> <div class="col-sm-4"> <p>Jestem kolumną 3</p> </div> </div> </div>
​

Komponenty

Bootstrap oferuje różnorodne komponenty, takie jak przyciski, karty, alerty, które można łatwo wdrożyć na stronie.
Przykład przycisku:
HTML
Copy
<button type="button" class="btn btn-success">Zielony Przycisk</button>
​
Przykład karty:
HTML
Copy
<div class="card" style="width: 18rem;"> <img src="..." class="card-img-top" alt="..."> <div class="card-body"> <h5 class="card-title">Tytuł karty</h5> <p class="card-text">Opis zawartości karty.</p> <a href="#" class="btn btn-primary">Przejdź dalej</a> </div> </div>
​

Klasy Pomocnicze (Utility Classes)

Klasy pomocnicze w Bootstrapie pozwalają na szybkie formatowanie i dostosowywanie stylów bez konieczności pisania dodatkowego CSS.
Przykład klasy tekstowej:
HTML
Copy
<p class="text-primary">Ten tekst jest w kolorze głównego motywu.</p>
​
Przykład klasy marginesu:
HTML
Copy
<p class="mt-5">Ten akapit ma margines górny 5 jednostek.</p>
​

Responsywność

Bootstrap automatycznie dostosowuje układ strony do różnych rozmiarów ekranu, co jest kluczowe dla współczesnych stron internetowych.
Przykład responsywnych kolumn:
HTML
Copy
<div class="row"> <div class="col-xs-12 col-md-8">.col-xs-12 .col-md-8</div> <div class="col-xs-6 col-md-4">.col-xs-6 .col-md-4</div> </div>
​
Inne elementy Bootstrapa wraz z przykładami można znaleźć na oficjalnej stronie projektu:
ALT

Ćwiczenie 1: Podstawy Bootstrapa

Utwórz nowy plik HTML i dodaj do niego podstawowy szkielet dokumentu.
Dołącz Bootstrapa do projektu, korzystając z linku CDN znajdującego się na oficjalnej stronie Bootstrap.
Znajdź dokumentację dla kontenera .container i .container-fluid i zastosuj je w swoim projekcie.

Ćwiczenie 2: Układ siatki (Grid Layout)

Przeczytaj dokumentację dotyczącą systemu siatki Bootstrapa.
Stwórz prosty układ strony z nagłówkiem, stopką i trzema kolumnami w sekcji głównej.
Zastosuj różne klasy siatki, aby układ był responsywny.

Ćwiczenie 3: Komponenty

Wybierz trzy komponenty Bootstrapa, takie jak przyciski, karty lub formularze.
Dodaj wybrane komponenty do swojej strony i dostosuj je według własnych preferencji.

Ćwiczenie 4: Projektowanie nawigacji

Utwórz pasek nawigacyjny z logo i kilkoma linkami.
Upewnij się, że nawigacja jest responsywna i dostosowuje się do różnych rozmiarów ekranu. Można to zrobić korzystając z narzędzie developerskich przeglądarki.

Ćwiczenie 5: Tworzenie Strony Kontaktowej

Użyj formularzy Bootstrapa, aby stworzyć prostą stronę kontaktową.

Zadanie dla ambitnych:

Przy pomocy bootstrapa stwórz responsywny frontend aplikacji, służącej do rezerwacji stolika w Twojej restauracji.
